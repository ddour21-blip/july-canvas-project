# July Canvas Export

이 프로젝트는 Vercel, Netlify, GitHub Pages 등에 즉시 배포할 수 있도록 구성된 표준 Vite + React 템플릿입니다.

## 배포 방법

1. 압축을 해제합니다.
2. 터미널에서 `npm install` 을 실행합니다.
3. **중요**: 제미나이 화면 우측 상단의 **[콘텐츠 복사]** 아이콘을 클릭하여 전체 코드를 복사한 뒤, 이 프로젝트의 `src/App.jsx` 파일에 덮어씌웁니다.
4. 코드를 GitHub 레포지토리에 푸시(Push)합니다.
5. Vercel에서 해당 레포지토리를 연결하여 즉시 배포합니다! (Firebase 데이터베이스와 자동 연동됩니다)
